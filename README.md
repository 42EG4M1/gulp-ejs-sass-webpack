# gulp-ejs-sass-webpack
Gulp + ejs boilerplate for my basic project.  
(ejs Starter)


## Require
* Node.js


## Getting Started
### Clone
    $ git clone git@github.com:42EG4M1/gulp-ejs-sass-webpack.git [projectName]

### Install npm packages
    $ npm i

### Build

    // gulp task
    $ npm run gulp


### Run only when New or Changed

    // svg sprite
    $ npm run svg

    // favicon
    $ npm run favicon
